# Level

## Layout
Size: 16*14

Template
```
================
=              =
=              =
=              =
=              =
=              =
=              =
=              =
=              =
=              =
=              =
=              =
=              =
================
```

Example
```
================
================
===           ==
===M  i  F II ==
====I ======= ==
============= ==
===        == ==
===        == ==
===     =   = ==
====   ===    ==
====   ==   = ==
====FFF===   F==
================
================
```
```
M: player
i/I: ice, if is lowercase, it's not fixed with other objects suchas wall, ice and so on.
F: fire
```
